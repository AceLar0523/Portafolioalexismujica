Controlador de Dron con ESP32

Descripción Breve

Diseño de circuitos y programación de microcontroladores para el control de vuelo y estabilización de un vehículo aéreo no tripulado (Dron).

Especificación Detallada

Integración de módulos giroscópicos y acelerómetros para la lectura de datos en tiempo real. Se programó un filtro PID (Proporcional, Integral, Derivativo) en C++ para estabilizar el vuelo ajustando la velocidad de los 4 motores de forma independiente. El circuito fue diseñado para ser liviano utilizando un ESP32 como procesador principal debido a su doble núcleo y bajo consumo.

Tecnologías Usadas

C++ ESP32 / PIC Electrónica PID Control

Generador de Assets (Pixel Art)

Descripción Breve

Script de automatización para la creación y modificación por lotes de assets visuales en formato pixel art para videojuegos.

Especificación Detallada

Se desarrolló un conjunto de scripts utilizando Python y la librería Pillow (PIL) para procesar directorios completos de imágenes. El programa permite reescalar sin pérdida de nitidez (Nearest Neighbor), aplicar paletas de colores específicas de forma automática y generar "spritesheets" a partir de imágenes individuales, reduciendo el tiempo de trabajo del artista en un 80%.

Tecnologías Usadas

Python Pillow (PIL) Automatización
